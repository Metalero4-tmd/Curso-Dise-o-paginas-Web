Tipo de Memorias:

MEMORIA RAM

RAM( Memoria de Acceso Aleatorio) Función: Almacena temporalmente datos y programas que la CPU está usando activamente para acceso rápido. Volatilidad: Volátil (los datos se pierden al apagar). Velocidad: Muy rápida. Ejemplos: Módulos DDR4, DDR5.

MEMORIA ROM

La memoria ROM (Read-Only Memory o Memoria de Solo Lectura) es un tipo de almacenamiento no volátil que guarda permanentemente instrucciones esenciales para el arranque y funcionamiento básico de dispositivos electrónicos
