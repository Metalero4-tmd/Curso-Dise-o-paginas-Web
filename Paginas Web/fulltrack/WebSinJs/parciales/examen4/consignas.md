## Trabajo Práctico Integrador – HTML y CSS

### Objetivo general

Desarrollar una **página web estática** utilizando **únicamente HTML y CSS** (sin JavaScript, sin frameworks, sin librerías externas), aplicando **todos los temas vistos hasta el momento**.

El objetivo no es memorizar, sino **demostrar comprensión** de la estructura HTML y del uso correcto de CSS para dar estilo y organización visual.

---

### Condiciones obligatorias

* Usar **solo HTML y CSS**.
* No usar JavaScript.
* No usar Bootstrap, Tailwind ni librerías externas.
* El CSS debe estar en un **archivo externo** (`style.css`).
* El HTML debe estar correctamente **indentado y ordenado**.

---

## Consigna

Crear un sitio web simple para una **marca / proyecto personal ficticio** (por ejemplo: un emprendimiento, un curso, una banda, una tienda pequeña, un portfolio, etc.).

Vos elegís el tema, pero **la estructura y los elementos pedidos son obligatorios**.

---

## Estructura obligatoria del HTML

El archivo `index.html` debe contener:

1. **Estructura básica HTML**

   * `<!DOCTYPE html>`
   * `<html>`, `<head>`, `<body>`
   * `<title>`
   * `<meta charset="UTF-8">`
   * Enlace al archivo CSS con `<link>`

2. **Header (encabezado)**

   * Un título principal (`h1`)
   * Un subtítulo (`h2` o `p`)

3. **Menú de navegación**

   * Una lista (`ul > li > a`)
   * Al menos **3 links** (no importa si no llevan a otra página)

4. **Sección principal (`main`)**
   Debe contener **al menos 3 secciones**:

   * **Sección 1: Presentación**

     * Un título (`h2`)
     * Un párrafo descriptivo
     * Una imagen (`img`)

   * **Sección 2: Contenido**

     * Un título
     * Un texto
     * Una **lista** (ordenada o desordenada)

   * **Sección 3: Tabla**

     * Un título
     * Una **tabla** con:

       * `table`, `tr`, `th`, `td`
       * Al menos 3 filas de datos

5. **Footer (pie de página)**

   * Texto de cierre (por ejemplo: copyright, autor, año)

---

## Requisitos obligatorios de CSS

El archivo `style.css` debe incluir y aplicar correctamente:

1. **Selectores**

   * Selector de etiqueta
   * Selector de clase

2. **Modelo de caja (Box Model)**

   * `margin`
   * `padding`
   * `border`

3. **Colores**

   * `color`
   * `background-color`

4. **Tipografía**

   * `font-family`
   * `font-size`
   * `font-weight`
   * `text-align`

5. **Fondos**

   * Uso de `background` o `background-image` en al menos una sección

6. **Layout básico**

   * Uso de `display` (block / inline / inline-block)
   * Uso de `width` y `height`

7. **Tablas estilizadas**

   * Bordes visibles
   * Espaciado interno (`padding`)
   * Bordes colapsados (`border-collapse`)

---

## Qué se evalúa

* Uso correcto de HTML semántico básico
* Aplicación coherente de CSS
* Orden y legibilidad del código
* Que **todo lo pedido esté presente**
* Que no haya propiedades o etiquetas que no se hayan visto

---

## Entrega

* `index.html`
* `style.css`

Ambos archivos deben funcionar correctamente al abrir `index.html` en el navegador.
